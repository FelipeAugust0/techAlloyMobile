Tech Alloy – Controle de Produção

Primeiro aplicativo mobile simples desenvolvido em aula com React Native (Expo) para controle de ocorrências industriais e cálculo de carga metálica (Cromo e Níquel).

Funcionalidades

- Registro de ocorrências;
- Campo de PIN de segurança;
- Cálculo automático do total de carga metálica;
- Interface simples;
- Exibição do peso total formatado;

Tecnologias Utilizadas

- React Native;
- Expo;
- JavaScript;
- React Hooks (useState);
- StyleSheet para estilização;

Como Executar o Projeto?
1️⃣ Clonar o repositório
git clone https://github.com/seu-usuario/tech-alloy.git

2️⃣ Acessar a pasta
cd techAlloyMobile

3️⃣ Instalar dependências
npm install

4️⃣ Iniciar o projeto
npx expo start

Depois disso, abra no:
Expo Go (celular) ou através do navegador em "http://localhost:8081/"

O sistema:
- Converte vírgula para ponto (padrão brasileiro → padrão JavaScript);
- Transforma os valores em float;
- Soma Cromo + Níquel;
- Exibe o resultado formatado com duas casas decimais;
